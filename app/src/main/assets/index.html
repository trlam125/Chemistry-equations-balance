Hướng dẫn lấy Google API key
  1. Mở trang Google AI Studio API Keys.
  2. Đăng nhập tài khoản Google và chấp nhận điều khoản nếu được hỏi.
  3. Nhấn Create API key, sau đó chọn hoặc tạo một Google Cloud project.
  4. Sao chép khóa, quay lại ứng dụng, dán vào ô phía trên và nhấn Lưu.
